## 🫀 Project 1 — Heart Disease EDA
**File:** `Pandas_Data_Analysis_Using_Heart_Disease_Dataset_MD_Mahfuz_Karim_Nasim.ipynb`

### Key Steps

- Dataset Structure and Missing Value Analysis
- Categorical Data Analysis
- Data Selection Using iloc
- Data Filtering Using Conditions
